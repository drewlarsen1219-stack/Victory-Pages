"Victory" represents and is symbolic for Jesus Christ victory over sin, death, and the devil.

"Paths" represents and is symbolic for this websites pathways offered to the user as an effort in being victorious like Christ is except over worldly things like finances, health and family. In a world where it is so unclear on which paths to take and what to prioritize

Victory Paths is an attempt to fill that gap by making the corrections using the wisdom made available to the author from God the Father, His begotten Son Jesus Christ and the Holy Spirit. Amen.

How all of this works

The new user will answer questionnaires that determine where the user currently stands in regards to life XP on the Skills Tracker.

*Website action: generate a list of questions like: "How many days per week to you excercise?" *

> The user will answer something like: "3 days per week."

*Website action: Users Skills Tracker updates*

*Website action: New Pathway(s) unlocked*

*Website action: The user now has new Pathways (IRL & online) quest available to complete.*

The existing User will be able to enter in their saved data from the progress that they have made already.

Continue completing Pathways and saving progress.

Once a user finished a Pathways then they will update the Skills Tracker.

*Website action: New quest is unlocked.*

Repeat process to progress further through the game.

A public forum is available for users which is the home page called "Victory Pages."
