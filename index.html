Q & A

What is "Victory"?

Symbolizes Jesus Christ's triumphant victory over sin, death, and the devil.

What is "Paths"?

The routes offered to users to achieve their own victory over worldly struggles (finances, health, family).

"Victory Paths"?

Provides clear, prioritized guidance drawn from the Bible, affirming that God's wisdom is the correct and intended methodology for our lives.

New User Process

An onboarding process where the new user answers questionnaires designed to assess their current standing (life XP) in various areas.

Example Question: "How many days per week do you exercise?"

Example Answer: "3 days per week."

Website Action: The user's Skills Tracker is updated with the new data.

Website Action: New Pathway(s) are unlocked based on the user's current skill level.

The user now has new Pathways (quests) available to complete, both IRL (In Real Life) and online.

Existing User Flow

Existing users can enter their saved data to reflect the progress they have already made.

Users continue completing Pathways and saving their progress.

Once a user finishes a Pathway, they update the Skills Tracker.

Website Action: A new quest is unlocked.

The user repeats this process to progress further through the game.

Public Forum

A public forum called "Victory Pages" is available for users and serves as the website's home page.
